# equity-risk-factor-modelling
Python-based analysis of equity returns using CAPM, Fama-French factor models, and ARIMA forecasting for financial risk assessment.

Equity Risk Factor Modelling

Overview

This project analyses Oracle Corporation (ORCL) stock returns using financial econometric techniques. The objective is to evaluate systematic risk exposures, investigate asset pricing relationships, and forecast return dynamics.

Techniques Used

* Python
* pandas
* NumPy
* statsmodels
* yfinance
* matplotlib

Methodology

Data Collection

* Oracle (ORCL) stock prices
* S&P 500 market data
* Treasury Bill rates
* Fama-French factor datasets

Asset Pricing Models

* CAPM
* Fama-French Five-Factor Model
* Momentum Factor Analysis

Time Series Analysis

* ADF Stationarity Testing
* KPSS Testing
* ARIMA Modelling
* BIC Model Selection

Forecast Evaluation

* Dynamic Forecasting
* RMSE Comparison
* Residual Diagnostics

Key Findings

* Equity returns exhibit non-normal behaviour and volatility clustering.
* Multi-factor models provide improved explanatory power relative to single-factor models.
* ARIMA models were used to generate out-of-sample forecasts and evaluate predictive performance.
